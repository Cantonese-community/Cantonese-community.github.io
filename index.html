粤语编程语言系咩? 佢系一门用粤语嚟同计算机沟通嘅编程语言。喺呢个语言度，计算机可以读明你写嘅粤语。所以话，你可以用粤语嚟操作(虐待)计算机。

點樣安裝?

pip install cantonese
